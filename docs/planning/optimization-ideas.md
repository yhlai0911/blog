# 功能優化建議

## 效能
- 文章與列表使用 ISR 或 tag-based revalidation。
- 瀏覽數防重複計數，並忽略 bot 與 prefetch。
- 封面圖使用 CDN 或自訂 image loader。

## SEO 與成長
- 自動產生文章 OG 圖。
- 根據標籤/分類顯示相關文章。
- 加入麵包屑結構化資料。

## 編輯流程
- 草稿自動儲存與版本紀錄。
- 標籤/分類批次調整。
- 未發佈文章預覽模式。

## 社群與審核
- 垃圾內容評分與連結數限制。
- 支援封鎖 IP 或 Email。
- 新評論通知。

## 分析
- 追蹤搜尋使用量與外連點擊。
- 顯示熱門來源與 UTM 報表。

## 使用者體驗
- 閱讀進度條與卡片上的閱讀時間。
- 後台快捷鍵。
- 更完善的空狀態與錯誤訊息。
