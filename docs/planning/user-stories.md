# 使用者故事

## 訪客 / 讀者
- 身為訪客，我想快速看到精選與最新文章，方便我探索內容。驗收：首頁顯示精選與最新文章並可點入細節。
- 身為讀者，我希望文章頁載入快速且排版易讀，方便在手機與桌機閱讀。驗收：主要內容在目標時間內載入並具備良好可讀性。
- 身為讀者，我想用分類與標籤瀏覽，方便探索相關主題。驗收：分類與標籤頁可列出文章與數量。
- 身為讀者，我想透過搜尋框找文章，方便以關鍵字查詢。驗收：搜尋結果能列出匹配文章。

## 留言者
- 身為留言者，我想以姓名（必填）與 Email（選填）留言，方便參與討論。驗收：表單驗證必填欄位並顯示成功訊息。
- 身為留言者，我想回覆特定留言，方便加入討論串。驗收：回覆表單可提交並帶入 parentId。
- 身為留言者，我想知道留言正在審核，避免誤以為沒送出。驗收：提交後顯示審核提示。

## 管理員 / 編輯
- 身為管理員，我想新增、編輯、刪除文章，方便管理內容。驗收：後台可完整 CRUD，含分類與標籤。
- 身為管理員，我想發佈或取消發佈文章，方便控制可見性。驗收：切換後更新 published 與 publishedAt。
- 身為管理員，我想審核評論，避免垃圾留言。驗收：可批准、取消、刪除。
- 身為管理員，我想看到瀏覽與熱門文章，方便了解成效。驗收：儀表板顯示總量與熱門排行。
- 身為管理員，我想調整站點設定（品牌與 SEO 預設），方便維護一致性。驗收：設定頁可儲存並套用。
